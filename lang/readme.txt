Adaptable language files
========================

Summary
-------------------------------------
This repository includes the Adaptable official language files available in English (UK) and Spanish (Spain).


Installation
-------------------------------------
You can upload the files to your moodle site using FTP/SFTP. 
Notice that you must create the folders if needed.
